# English Self-Talk Learning App

A minimalist, high-impact web application designed to help Indonesian learners achieve English fluency through structured 5-minute daily self-talk routines.

## 🚀 Key Benefits & Features

- **Fluency via Self-Talk**: Practice natural English by thinking and speaking in routines that mirror daily life.
- **Interactive Deep Learning**:
    - **Toggle Translation**: Switch between Indonesian and English to verify comprehension.
    - **Phonetic Pronunciation**: Integrated guides to help you speak with confidence.
    - **Smart Sentence Breakdown**: Automated semantic parsing that highlights key vocabulary and their Indonesian meanings.
- **Progress Tracking**: Personal local tracking to mark and monitor sentences you have mastered (*"Tandai Sudah Lancar"*).
- **Embedded Vocabulary**: Every sentence is built upon specific vocabulary headers, ensuring you learn words in context.
- **Admin & Multi-User Support**: Robust management interface for content creation, user oversight, and shared learning pools.
- **Archival System**: Organize your learning journey by year and month.
- **Mobile Optimized**: A sleek, premium, and responsive UI built for learning on the go.

## 🛠️ Technology Stack
- **Backend**: PHP (PDO)
- **Frontend**: Vanilla JS, Modern CSS (Inter/Google Fonts)
- **Database**: MySQL
